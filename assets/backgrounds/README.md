# Background Assets

This directory contains optional background images used by GENtle windows.

- `Synthetic_Biology.png` is a generated bitmap added on 2026-06-05 for a
  future synthetic-biology window. Prompt summary: a legally distinct,
  cheerful puppet-like chef scientist cooking a playful molecular-biology
  experiment at a lab bench, with pipettes, tubes, agar plates, a thermocycler,
  electrophoresis box, plasmid maps, flasks, and negative space for UI overlay.
- `synthetic_biology_chef.png` is a Codex-generated alternate created on
  2026-06-05 with the same legally distinct lab-chef concept and preserved as
  an additional candidate background for future synthetic-biology UI work.
