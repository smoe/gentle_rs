# GENtle Mascot Assets

This folder collects exploratory and app-ready mascot assets for GENtle.

Current concept:

- `NZW`: white New Zealand White mouse, used as the careful provenance and
  documentation voice.
- `B6`: black C57BL/6J mouse, used as the reproducibility and re-run voice.

The mascot concept is optional presentation only. Core features such as Coffee
Break, external-context links, and agent/tool workflows should remain
headless, deterministic, and usable through shared engine/shell routes without
depending on image assets.

## App-Ready Assets

### `Mascot_transparent.png`

- Intended use: startup splash-screen mascot.
- Format: RGBA PNG with a real alpha channel.
- Current status: wired into the egui splash screen via
  `crates/gentle-gui/src/icons.rs`.

## Experiments

`experiments/` contains generated visual studies and should not be treated as
polished app UI without review. These images may have AI-generation artifacts
such as distorted lettering, pseudo-transparent backgrounds, or inconsistent
logos. Prefer cropping, retouching, or replacing them with vector/2.5D assets
before release use.

### `experiments/nzw_b6_pair_prompt1.png`

- Origin: generated by ChatGPT image generation from a user/Codex prompt on
  2026-05-23.
- Prompt intent: two upright anthropomorphic lab mice, a white NZW mouse and a
  black C57BL/6J mouse, wearing lab coats and small hats; clipboard labelled
  "Provenance" and tablet labelled "Re-run"; original mascot concept for
  GENtle.
- Intended use: exploration of a reusable mascot pair.
- Current status: exploratory raster asset, not yet wired into the app.
